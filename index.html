CSS Layout - The position Property

There are five position values in css:

Property Description
static HTML elements are static by default. Not affected by top, bottom, left, right properties. Always positioned according to normal flow. CSS: position: static;
relative An element with the relative position will be positioned "relative" to its normal (static) position. Top, right etc. properties will move it from its normal position. Other elements will not be adjusted to fill that gap. CSS: position: relative;
fixed An element with the fixed position is positioned relative to the viewport. It will always stay in the same position even if the webpage is scrolled. This is useful for elements like nav bars. CSS: position: fixed;
absolute An element with the absolute position is positioned relative to the HTML element it is inside of. If it is not inside any HTML elemnt it will be positioned relative to documnet body and will move along with page scrolling. CSS: position: absolute;
sticky An element with the sticky position is positioned based on users scroll position. It is relative until a given offset position in the webpage then it sticks in place. This does not work in Internet explorer and needs a special -webkit- prefix to work in safari. CSS: position: sticky;
